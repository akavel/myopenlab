RECOGE DATO (Librería FlowCharts-Variables)

 

Este bloque permite recoger un dato de la aplicacion y poner su valor dentro del diagrama de flujo haciendolo esto en todo momento. Las variables que permite son (bol, dbl, str)

La diferencia con el bloque "Leer Dato" es que este bloque lee el dato en todo momento, no cuando la ejecución del diagrama de flujo llega el bloque como ocurre con este bloque "Leer Dato"..
